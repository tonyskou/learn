# 📖 Μαθαίνω! — Οδηγίες Εγκατάστασης

## Εγκατάσταση σε Android (Chrome)

1. Άνοιξε το URL με **Chrome**
2. Πάτησε ⋮ → **"Προσθήκη στην αρχική οθόνη"**
3. Πάτησε **"Προσθήκη"**
4. Εμφανίζεται σαν κανονικό app, δουλεύει offline ✓

---

## Εγκατάσταση σε iPhone / iPad (Safari)

1. Άνοιξε το URL με **Safari** (όχι Chrome — το Safari μόνο υποστηρίζει εγκατάσταση στο iOS)
2. Πάτησε το κουμπί **Κοινοποίηση** (□↑) κάτω στην οθόνη
3. Σκρολ κάτω → **"Προσθήκη στην Αρχική Οθόνη"**
4. Πάτησε **"Προσθήκη"** πάνω δεξιά
5. Η εφαρμογή εμφανίζεται στην αρχική οθόνη σαν κανονικό app ✓

### Για τον φωνητικό συλλαβισμό στο iPad/iPhone:
Την πρώτη φορά που θα ανοίξεις την εφαρμογή εμφανίζεται ένα παράθυρο.
Πάτησε "Ξεκινάμε!" για να ενεργοποιηθεί η φωνή (απαίτηση του iOS).

---

## Τι περιέχει η εφαρμογή

- **70 σετ × 10 λέξεις = 700 λέξεις** (~2 χρόνια υλικό)
- Συλλαβισμός με πολύχρωμα γράμματα
- Επιβράβευση 🌟 στο τέλος κάθε σετ
- Φωνητικός συλλαβισμός κλειδωμένος 7 μέρες μετά την ολοκλήρωση
- Καρτέλα Επανάληψης και Στατιστικών
- Δουλεύει offline (χωρίς internet)
- Αποθηκεύει αυτόματα την πρόοδο
