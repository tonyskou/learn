# 📖 Διαβάζω! — Εφαρμογή Εκμάθησης Ανάγνωσης & Συλλαβισμού

Μια δωρεάν, offline PWA εφαρμογή για παιδιά που μαθαίνουν να διαβάζουν και να συλλαβίζουν στα ελληνικά. Με **700 λέξεις** σε 70 θεματικά σετ, φωνητικό συλλαβισμό, και **συγχρονισμό προόδου στο cloud** μέσω Firebase.

---

## 🚀 Γρήγορη Εγκατάσταση

### Ανέβασμα στο GitHub Pages

1. Φτιάξε ένα **δωρεάν** repository στο GitHub: `diabazo` (Public)
2. Ανέβασε όλα τα αρχεία από αυτόν τον φάκελο:
   - `index.html`, `words.js`, `manifest.json`, `sw.js`
   - `icon-180.png`, `icon-192.png`, `icon-512.png`
3. Πήγαινε **Settings → Pages → Branch: main → Save**
4. Το URL σου θα είναι: `https://[το-username-σου].github.io/diabazo/`

### Εγκατάσταση στις συσκευές

| Συσκευή | Πώς |
|---|---|
| **Android** | Άνοιξε το URL με Chrome → ⋮ → «Προσθήκη στην αρχική οθόνη» |
| **iPhone / iPad** | Άνοιξε το URL με **Safari** → Κοινοποίηση (□↑) → «Προσθήκη στην Αρχική Οθόνη» |

Μετά την εγκατάσταση, η εφαρμογή λειτουργεί **και χωρίς internet** 📴

## 🔐 Firebase — Συγχρονισμός & Ασφάλεια

Για να συγχρονίζεται η πρόοδος ανάμεσα σε πολλές συσκευές (π.χ. iPad + Android), χρειάζεται Firebase. Είναι **δωρεάν** και η εφαρμογή δουλεύει κανονικά και χωρίς αυτό (τοπικά).

### Τι προσφέρει το Firebase

- 🔐 **Σύνδεση γονέα** με email + κωδικό — μόνο εσύ βλέπεις τα δεδομένα
- 👧 **Πολλαπλά παιδιά** ανά λογαριασμό (π.χ. «Μαρία» και «Γιώργος»)
- ☁️ **Αυτόματος συγχρονισμός** — η πρόοδος είναι ίδια σε όλες τις συσκευές
- 🛡️ **Ασφάλεια** — κανένας άλλος δεν μπορεί να δει τα δεδομένα σου, ακόμα κι αν γράψει το ίδιο όνομα παιδιού

### Δομή δεδομένων

```
users/
  └── [το UID σου] (από το login)/
      └── children/
          ├── child_μαρία  → { πρόοδος της Μαρίας }
          └── child_γιώργος → { πρόοδος του Γιώργου }
```

Κάθε γονιός έχει το δικό του UID → τα δεδομένα είναι πάντα ξεχωριστά.

### Στήσιμο Firebase (10 λεπτά, μία φορά)

1. Πήγαινε στο **[console.firebase.google.com](https://console.firebase.google.com/)**
2. **Create a project** → `diabazo` → Continue → Create
3. Μετά τη δημιουργία, πάτα το **</>** (Web app) → `diabazo` → Register
4. **Αντίγραψε** το `firebaseConfig` που εμφανίζεται
5. **Άνοιξε το `index.html`** και αντικατάστησε το `firebaseConfig` (~γραμμή 320) με αυτό που αντέγραψες
6. **Authentication**: Build → Authentication → Sign-in method → **Email/Password → Enable** (μόνο το Email, ΟΧΙ το passwordless)
7. **Firestore Database**: Build → Firestore → Create database → `eur3 (europe-west)` → Standard edition → Start in test mode → Create

### Πρώτη χρήση με Firebase

1. Άνοιξε την εφαρμογή → εμφανίζεται η οθόνη σύνδεσης
2. Πάτα **«Δημιούργησε έναν»** → βάλε email + κωδικό (6+ χαρακτήρες)
3. Στις άλλες συσκευές: **Σύνδεση** με τα ίδια στοιχεία
4. Μετά τη σύνδεση, γράψε το όνομα του παιδιού (π.χ. «Μαρία»)
5. Η εφαρμογή συγχρονίζει αυτόματα!

## 📱 Οθόνες & Λειτουργίες

### Οθόνη Σύνδεσης
- **Σύνδεση** με email + κωδικό
- **Δημιουργία λογαριασμού** για πρώτη φορά
- **Συνέχεια χωρίς σύνδεση** (τοπική λειτουργία)

### 📚 Μάθηση (κύρια οθόνη)
- 70 θεματικά σετ (Οικογένεια, Ζώα, Φαγητό, Σχολείο, κλπ.)
- 10 λέξεις ανά σετ — σύνολο **700 λέξεις**
- Πολύχρωμος συλλαβισμός: κεφαλαία χωρίς τόνους, **πεζά με τόνους**
- Πάτησε την κάρτα για να μαρκάρεις μια λέξη ως «την έμαθα»
- Μπάρα προόδου ανά σετ

### 🔤 ΑΒ — Αλφαβήτα & Γραμματική
- **Αλφαβήτα:** Και τα 24 γράμματα σε κεφαλαία & πεζά, με ελληνική ονομασία. Πάτησε για να ακούσεις την προφορά 🔊
- **Δίψηφα Φωνήεντα:** αι, ει, οι, υι, αυ, ευ, ου, ηυ — πώς προφέρονται με παραδείγματα
- **Δίψηφα Σύμφωνα:** μπ, ντ, γκ, γγ, τσ, τζ — πώς προφέρονται με παραδείγματα
- **Απλή Γραμματική:** 7 κανόνες γραμμένοι απλά για παιδί (τόνος, φωνήεντα, σύμφωνα, τελικό σίγμα, δίψηφα, συλλαβές)

### 🔊 Φωνητικός Συλλαβισμός
- Η φωνή (TTS στα ελληνικά) ξεκλειδώνει **7 μέρες μετά** την ολοκλήρωση του σετ
- Σκοπός: το παιδί πρώτα διαβάζει μόνο του και μετά ακούει για επιβεβαίωση
- Δύο κουμπιά: «🔊 Συλλαβές» (αργά, μία-μία) και «🔊 Λέξη» (ολόκληρη)

### 📖 Ιστορίες
- **30 μικρά παραμύθια** σε απλά ελληνικά για παιδιά
- **Κάθε λέξη είναι clickable** — πάτα όποια λέξη θέλεις και ακούς την προφορά της 🔊
- **«Άκου όλη την ιστορία»** — αυτόματη ανάγνωση πρόταση-πρόταση

### 🔄 Επανάληψη
- Όλα τα ολοκληρωμένα σετ εμφανίζονται εδώ
- Πάτησε οποιοδήποτε για επανάληψη
- Δείχνει αν η φωνή είναι ξεκλείδωτη ή πόσες μέρες απομένουν

### ⭐ Πρόοδος
- Σύνολο λέξεων που έμαθες
- Σετ που ολοκλήρωσες
- Ποσοστό συνολικής προόδου
- Αλλαγή παιδιού / προσθήκη νέου
- Έξοδος από τον λογαριασμό

## 🔧 Προχωρημένο: Κλείδωμα Firestore

Αφού επιβεβαιώσεις ότι όλα δουλεύουν, πήγαινε Firebase Console → Firestore → Rules και αντικατάστησε με:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```

Αυτό διασφαλίζει ότι κάθε χρήστης έχει πρόσβαση **μόνο** στα δικά του δεδομένα.
